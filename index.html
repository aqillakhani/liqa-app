Liqa

A private listing management tool for the Liqa Etsy shop

Private App — Single User

What is Liqa?

Liqa is a private, single-user Python application built to manage listings for the Liqa Etsy shop. It uses the Etsy Open API v3 to batch-upload and manage digital coloring book PDF listings, preview images, tags, and descriptions.

Features

  • Batch creation of digital download listings via the Etsy Open API v3
  • Upload digital files (PDFs) and listing preview images
  • Manage tags, descriptions, pricing, and categories in bulk
  • OAuth 2.0 with PKCE authentication flow

Who uses this app?

This app is used exclusively by the owner of the Liqa Etsy shop. It is not distributed to other sellers and does not collect or process any third-party user data. It runs locally on a single machine and only accesses the shop owner's own Etsy account.

API Endpoints Used

  • POST /v3/application/shops/{shop_id}/listings — Create listings
  • POST /v3/application/shops/{shop_id}/listings/{listing_id}/files — Upload digital files
  • POST /v3/application/shops/{shop_id}/listings/{listing_id}/images — Upload listing images
  • GET /v3/application/shops/{shop_id}/listings — Retrieve existing listings
