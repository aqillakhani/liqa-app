Privacy Policy

Last updated: March 24, 2026

Overview

Liqa ("the App") is a private, single-user application that manages listings for the Liqa Etsy shop. This privacy policy describes how the App handles data when interacting with the Etsy Open API v3.

Data Collection

The App does not collect, store, or process any personal data from third parties. Specifically:

  • No customer data is accessed or stored
  • No buyer information is collected
  • No analytics or tracking cookies are used
  • No data is shared with any third parties
  • No user accounts are created beyond the single shop owner

Etsy API Data

The App accesses the following data from the Etsy API, exclusively for the shop owner's own account:

  • Listing data: Titles, descriptions, tags, prices, and images for the shop owner's own listings
  • Shop data: Shop ID and shop sections for organizing listings
  • Digital files: PDF files uploaded as digital downloads for the shop owner's products

This data is used solely to create and manage the shop owner's own product listings and is not shared, sold, or disclosed to anyone.

Authentication

The App uses Etsy's OAuth 2.0 with PKCE flow for authentication. Access tokens are stored locally on the shop owner's machine and are never transmitted to any third-party server.

Data Storage

All data processed by the App remains on the shop owner's local machine. No cloud storage, external databases, or third-party services are used to store shop or listing data.

Data Retention

OAuth tokens are retained locally only for the duration needed to manage listings. No historical data is maintained beyond what is necessary for the App to function.

Children's Privacy

The App does not collect any data from children or any other users. It is a private tool used exclusively by a single adult shop owner.

Changes to This Policy

This privacy policy may be updated if the App's functionality changes. Any updates will be reflected on this page with a revised date.
